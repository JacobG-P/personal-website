GYM 2024 Data

I’ve been going to the gym for a while now and have been fairly consistent this past year, using Google Sheets to track my progress. This page hosts my 2024 Summation Google Sheet, which provides a comprehensive overview of my exercises, including personal bests and totals. I record weights and reps for all working sets in the respective monthly Google Sheet journal, from which the summation sheet pulls data to display a complete summary so far.

I want to expand and enhance the data for next year’s tracking. However, I’m unsure whether adding more features will be effective or actually utilized.

This gym tracking system also includes daily information about weight, sleep, calories, and water, which I tracked consistently during a bulking stage. However, I’ve since stopped keeping up with it. There are also features for tracking workout durations, but I always forget to clock out.

Title Definitions

Exercise Naming Convention

  • Exercise: The type of exercise (e.g., row, squat, etc.).
  • Variation: Specific variations (e.g., single arm, isometric, supinated, etc.).
  • Weight: Type of weight used (e.g., cable, dumbbell, etc.).
  • Body: Focused muscle group in compound exercises, if specified (e.g., lat, quad, hamstring, etc.).
  • Equipment: Tools or attachments used (e.g., handle, rope, etc.).
  • Machine: Machine type, if applicable (e.g., different pulley setups or plate vs. pin-loaded). This category is specific to my gym.

Total Statistics

  • Total Volume: Total weight lifted (weight x reps) for that exercise.
  • Total Sets: Total number of sets completed for that exercise.
  • Total Frequency: Number of times the exercise appeared in workouts.

Max Statistics

  • Max Volume: Highest weight x reps achieved.
  • Max Weight: Heaviest weight lifted.
  • Reps: Number of reps performed at the heaviest weight.
  • Date: Most recent date of achieving a personal best.
  • Date Frequency: Number of personal bests achieved on the same date.
  • Max Weight Frequency: Number of times the max weight was achieved.
  • Dates: All dates on which lifts occurred.

Daily Information

  • Body Weight: Recorded morning weight.
  • Sleep: Hours slept.
  • Calories: Total daily caloric intake.
  • Water: Estimated daily water consumption.

Monthly Information

  • Month: The month in which the lift occurred.
  • Month Frequency: Number of personal bests during the month.

Weekly Information

  • Week: The week in which the lift occurred.
  • Week Frequency: Number of personal bests during the week.
